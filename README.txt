Thank you for downloading! To install, drag both .wotmod files from the option folder(s) of your choice to <Your WoT Directory\mods\<Game Version>

(Example: C:\Games\World_of_Tanks\mods\0.9.19.0.2)

Note: the scripts file will remove player emblems from the turret and the clan icon from the hull.
